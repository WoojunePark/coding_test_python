# 문제를 읽으면 생각해야 하는 것
+ 명시적 조건 (explicit)
    + 시작
    + 끝
+ 암시적 조건 (implicit)
    + minimum : 
        + e.g. 길이가 0인 회의도 가능? 0과 0을 비교하는 상황은 없나?
    + maximum : 
        + e.g. 2^-31
+ 부등호 (이항연산자)
    1) 같은 원소에 대해서는 무조건 F
    2) 좌우 바꾸면 무조건 F
    3) 삼단논법 가능해야
    4) 0 처리
        + 둘 중에 하나가 0이면 항상 T/F 고정 등
        + 혹은 아예 0은 입력에서 제외
    
    + 팁 : a/b < c/d 대신, a\*d < b\*c
